## Run Tests

### Run via IDE
Specify config in VM Options:
```
-Dconfig=selendroid.emu.api19
```

### Run via commandline

Run calculator tests on configuration specified in calc.emu.api23.properties:

```
./gradlew clean calcTests -Pconfig=calc.emu.api23
```
